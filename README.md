# What is Terraform?
Terraform is an open-source Infrastructure as Code (IaC) tool developed by HashiCorp.
Terraform allows us to create, manage, update, and destroy cloud infrastructure using code instead of manual configuration.

Terraform supports multiple platforms including:
- AWS
- Azure
- Google Cloud Platform (GCP)
- Kubernetes
- Docker
- Oracle Cloud
